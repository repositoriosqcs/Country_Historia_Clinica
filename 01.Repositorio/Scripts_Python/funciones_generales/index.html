    Funciones Generales¶

    EL objetivo de este script es consolidar las funciones que son transversales para las ETLs del repositorio y el tablero. En donde se destacan las cadenas de conexión a las bases de datos y el listado de tablas a incluir.

    Librerías¶

    Para empezar, se importan las librerías estandar de python y que vienen incluidas por defecto

    #Librerias Estandar
import os
import time
from datetime import datetime, date, time as datetime_time
import decimal
    Luego se importan las librerías externas que se hacen parte del entorno virtual explicado anteriormente.

    #Librerias Externas

    Cadenas de conexión¶

    De esta manera se definen las cadenas de conexión de origen y destino. Es importante tener en cuenta, que al momento de hacer la instalación se deben configurar las cadenas con las rutas adecuadas.

    # Diccionario de conexiones
connections_start = {
    "CTY_PRI": r"DRIVER={SQL Server};SERVER=QCS-CONS04;DATABASE=CTY_PRI;Trusted_Connection=yes;",
    "COL_CTY": r"DRIVER={SQL Server};SERVER=QCS-CONS04;DATABASE=COL_CTY;Trusted_Connection=yes;",
}

connections_final = {
    "CTY_PRI_PARAMETRICA": r"DRIVER={SQL Server};SERVER=QCS-CONS04;DATABASE=CTY_PRI_PARAMETRICA;Trusted_Connection=yes;",
    "COL_CTY_PARAMETRICA": r"DRIVER={SQL Server};SERVER=QCS-CONS04;DATABASE=COL_CTY_PARAMETRICA;Trusted_Connection=yes;",
}

    Funciones Transversales¶

    Así, se define la siguiente función que permite identificar la conexión de la base de origen con la base de destino, es decir, la base original y la base paramétrica: 

    def db_final(db_name):
    if db_name == "COL_CTY":
        db_name_final ='COL_CTY_PARAMETRICA'
    elif db_name == "CTY_PRI":
        db_name_final ='CTY_PRI_PARAMETRICA'
    return db_name_final
    Finalmente, se define la siguiente función que permite obtener el listado de tablas de la base paramétrica sin tener en cuenta las tablas asociadas a los metadatos o indicadores de calidad: 
    def lista_tablas(db_name_final,cursor):
    query_tables_list = f"""SELECT 
            TABLE_NAME
            FROM        
        {db_name_final}.INFORMATION_SCHEMA.TABLES
        WHERE 
        TABLE_TYPE = 'BASE TABLE' 
        AND TABLE_CATALOG = '{db_name_final}'
        AND TABLE_NAME NOT IN ('Metadata_Tablas','Metadata_Campos','Metadata_Relaciones','Fact_Metadatos_Tablas','Fact_Metadatos_Campos','Fact_Indicadores_Historicos','Dim_Componente_Calidad');"""
    # Ejecutar la consulta
    cursor.execute(query_tables_list)

    # Obtener los resultados de la consulta
    rows = cursor.fetchall()

    # Extraer los nombres de las tablas en una lista
    tables_list = [row[0] for row in rows]

    return tables_list
